## üîÑ Loop Architecture

### 5-Level Infinite Loop Structure

1. **Level 1**: Top-level loop (`if __name__ == "__main__"`)
   - Ultimate safety net
   - Restarts entire bot on crashes

2. **Level 2**: Main async function (`main()`)
   - Restarts bot instance if `start_bot()` returns

3. **Level 3**: Start bot function (`start_bot()`)
   - Manages async tasks
   - Restarts tasks if they complete

4. **Level 4**: Trading loop wrapper (`_run_trading_loop_forever()`)
   - Wraps `trading_loop()` and restarts if it returns

5. **Level 5**: Main trading loop (`trading_loop()`)
   - Core trading logic
   - Runs continuously

**Result**: Bot runs forever until manual stop (Ctrl+C)

   - Check position management settings

   - Use GPU acceleration
